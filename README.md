# Nuxt Starter

Nuxt Starter is a template for creating a Nuxt project. It is a collection of tools and libraries.

## Features

- Nuxt 3
- Nuxt UI Pro v3
- nuxt-auth-utils
- nuxt-content
- nuxt-og-image
- nuxt-test-utils
- nuxt-ui-pro
- nuxt-icon
- nuxt-fonts
- nuxt-image
- drizzle-orm
- nuxthub
- tailwindcss
- typescript
- vitest
